# `base32-encoding `

>

## Usage

```js
var  = require('base32-encoding')
```

## API

### `base32.encode(buf, [output], [offset])`

### `base32.decode(buf, [output], [offset])`

### `base32.encodingLength(buf)`

### `base32.stringify(buf, [alphabet])`

### `base32.parse(buf, [alphabet])`

## Install

```sh
npm install base32-encoding
```
